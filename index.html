The Challenge

The Strategy

The main focus of redesigning the website is making it simpler. The main goals are to remove sections that are not being viewed very often, removing the fixed text, and reassign spacing between content.

The Solution

The website will focus mainly for MICA Graphic design students. It will also serve as a portfolio site for students and employers. This site can be informative for incoming MICA students as well.

Simplifying the layout is the aim for redesigning. After interviewing users of the website, many share same concerns. Most of them focus only on section of news, internship, and student profile. Some sections of the website were not visited by anyone previously before the interview.

The “Graphic Design” fixed text is removed. This element distracted user experiences. Sections are redesigned. Some less visited sections are removed. Navigation system becomes more collapse. Color palettes are slightly changed, using more subtle colors.

By simplify the general layout and remove distracted elements creates a more user friendly and focal environment for acting as a informative and student portfolio site.
